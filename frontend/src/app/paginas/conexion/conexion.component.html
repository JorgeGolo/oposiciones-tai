Conexión con el Backend (ASP.NET Core)

Configuración esencial para comunicar nuestro frontend de Angular con la Web API en desarrollo y producción.

1. Habilitar CORS en el Backend

Imprescindible para que el navegador no bloquee las peticiones. Añade esto en el Program.cs de tu API: 

builder.Services.AddCors(options =>
{
    options.AddPolicy("FrontendPolicy", policy =>
    {
        policy.WithOrigins(
                "http://localhost:4200",
                "https://api-oposicion-tai-jorgegl.azurewebsites.net"
              )
              .AllowAnyHeader()
              .AllowAnyMethod();
    });
});

2. Configurar Entornos en Angular

Genera los archivos de entorno ejecutando ng generate environments y configura tus URLs:

Desarrollo (environment.development.ts)

export const environment = {
  production: false,
  apiUrl: 'https://localhost:5026'
};

Producción (environment.ts)

export const environment = {
  production: true,
  apiUrl: 'https://api-oposicion-tai-jorgegl.azurewebsites.net'
};
